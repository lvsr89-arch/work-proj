Цель:
Практика в работе с WebDriver и Библиотеками Log2j2, web driver manager.


Описание/Пошаговая инструкция выполнения домашнего задания:
Что будем тестировать? — приложение https://wishlist.otus.kartushin.su/wishlists/ предоставляет возможность ведения списков желаний

Требования к фреймворку:

Java
Maven/Gradle
TestNG/Junit
Настроено логирование
Код проекта хранится в Git
Для работы со страницами используется паттерн Page Object
Код оформлен согласно Java Code Conventions, комментарии в стиле Javadoc (http://codenet.ru/webmast/java/JavaDoc/) приветствуются
(*) Реализована возможность кроссбраузерного тестирования и удаленного запуска тестов
Разрешается подключение других библиотек, использование BDD-подхода, Spring, Lombok.

Необходимо выделить 4-5 критичных функциональностей и автоматизировать их тестирования


Критерии оценки:
Баллов всего 10, проходной балл 10.
+3 балла за каждое задание.
+1 балл за использование log4j2, webdriver manager.
